HiSky – Flare Output Instructions

Using your HTML5 template

To set up and edit your home page:

  1. Open Content/Home.htm.

  2. To change the text above the home search bar, edit the first line of text.

  3. To change the hyperlinks of the home tiles, right click on the tile’s hyperlink and choose Edit Hyperlink.

  4. To add a tile:

    1. Click on an li tag to select an existing tile. Copy the tile.

    2. Press Enter to start a new line, and paste the tile into the new line. The structure should remain like a regular bulleted list.

    3. Edit the new tile hyperlink using step 3 above.

    4. To delete a tile: Select the li tag of tile and press Delete.

To build your HTML5 output:

  1. Prepare content for your output and build a TOC with the correct structure for the output.

  2. Check your TOC is conditioned properly so that any print pages are conditioned with the print only tag, and any web only pages are conditioned with the screen only tag.

  3. Make a copy the existing sample HTML5 target in Project/Targets. All the settings in this target ensure that the template design is applied.

  4. In the General tab of the target, change the Primary TOC to the new TOC.

  5. In the Conditions tab, ensure you have the right conditions included and excluded.

  6. In the Variables tab, update all the variables values as required for the new output.

  7. Click Build.

Using your PDF template

To build your PDF output:

  1. Prepare content for your output.

  1. Make a copy of the existing Sample TOC.

  2. Rename the copied TOC, remove existing content except for the cover, notice, table of contents, and back cover topics. Add your content to the TOC, before the back cover, in the correct structure.

  3. Check your TOC is conditioned properly so that any print pages are conditioned with the print only tag, and any web only pages are conditioned with the screen only tag.

  4. Make a copy the existing sample PDF target in Project/Targets. All the settings in this target ensure that the template design is applied.

  5. In the General tab of the target, change the Primary TOC to the TOC required for the new output.

  6. In the Conditions tab, ensure you have the right conditions included and excluded.

  7. In the Variables tab, update all the variables values as required for the new output. These variables control the details on the cover and in the page headers.

  8. Click Build.

Table styles

One table style has been created – it is called Default. When you create a table, select this style to get a colored header and predefined text spacing.
